# 🧠 Mindbug

Cette application web a été conçue pour les joueurs du jeu de cartes **Mindbug**. En voyage, oubliez les jetons : suivez vos points de vie directement sur votre téléphone.

---

## 📱 Installation et Utilisation

### 🌐 Via le navigateur
L'application est hébergée sur **GitHub Pages**. Pour l'utiliser [rendez-vous ici](https://aboivert.github.io/Mindbug/) !

### 🤖 Via le fichier APK
Pour installer l'application directement sur votre téléphone Android, suivez la procédure suivante :
1. Transférez [le fichier .apk](https://github.com/aboivert/Mindbug/blob/main/android/app/build/outputs/apk/debug/app-debug.apk) sur votre Android.
2. Autorisez l'installation depuis des "sources inconnues" si demandé.
3. Suivez les instructions d'installation classiques.
